# Introduction

In today’s competitive E-commerce landscape, understanding customer behavior and predicting future purchases are critical for driving business growth. This project dives into a rich transactional dataset from a UK-based online retail store, spanning December 2010 to December 2011. The dataset captures over 25,000 transactions from n customers, offering a treasure trove of insights into purchasing patterns, customer preferences, and product performance.

The primary goal of this analysis is twofold:
1. **Explore and Understand**: _Uncover hidden trends in customer behavior, product categories, and regional sales performance._
2. **Predict and Act**: _Develop a machine learning model to predict the purchasing behavior of new customers based on their initial interactions with the store. This predictive capability can empower the business to tailor marketing strategies, optimize inventory, and enhance customer satisfaction._

By combining exploratory data analysis (EDA), customer segmentation, and predictive modeling, this project aims to deliver actionable insights that can drive strategic decision-making for the online retail business.

# Dataset Overview

The dataset, sourced from the UCI Machine Learning Repository, contains transactions from a UK-based online retail store specializing in unique all-occasion gifts. Here’s a quick look at the dataset schema:
| **Column Name** |	**Description** |
| ----------- | ----------- |
| InvoiceNo	| Unique identifier for each transaction (25,900 unique values) |
| StockCode	| Unique identifier for each product (4,070 unique values) |
| Description |	Product description (4,224 unique values) |
| Quantity	| Number of units purchased in the transaction |
| InvoiceDate |	Date and time of the transaction |
| UnitPrice	| Price per unit of the product |
| CustomerID | Unique identifier for each customer |
| Country	| Country where the transaction occurred |
